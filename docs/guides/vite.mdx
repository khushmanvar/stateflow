You can use the plugins from the `stateflow/babel` bundle to enhance your developer experience when using Vite and Stateflow.

In your `vite.config.ts`:

```js
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import stateflowDebugLabel from 'stateflow/babel/plugin-debug-label'
import stateflowReactRefresh from 'stateflow/babel/plugin-react-refresh'

// https://vitejs.dev/config/
export default defineConfig({
  plugins: [
    react({ babel: { plugins: [stateflowDebugLabel, stateflowReactRefresh] } }),
  ],
  // ... The rest of your configuration
})
```
